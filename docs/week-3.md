# Week 3: Dataset Preprocessing, Labelling and DCGAN

## Objective

The objective of this week was to improve data quality through preprocessing and implement a Deep Convolutional GAN (DCGAN) for better image generation.

## What I Learned

* Image normalization and preprocessing techniques.
* Dataset organization and labeling strategies.
* DCGAN architecture and convolutional layers.
* Stable GAN training practices.

## Implementation

* Preprocessed anime face images.
* Explored labeling methods based on visual attributes such as hair color and eye color.
* Implemented a DCGAN architecture using convolutional layers.
* Trained the DCGAN on the anime face dataset.

## Results

* Generated higher-quality images compared to the vanilla GAN.
* Improved feature learning through convolutional networks.
* Better visual consistency in generated anime faces.

## Challenges

* Selecting useful labels for future conditional generation.
* Increased computational requirements.
* Hyperparameter tuning for stable training.
