# Week 2 - Vanilla GAN on Anime Face Dataset

## Objective

Train a basic Generative Adversarial Network (GAN) on anime face images to understand adversarial learning.

## Dataset

Anime Face Dataset from Kaggle.

## Architecture

- Generator
- Discriminator
- Binary Cross Entropy Loss
- Adam Optimizer

## Results

The generator gradually learns to create anime-style faces from random noise.

## Learning Outcomes

- Understanding GAN fundamentals
- Adversarial training
- Image generation using PyTorch
